# Wander Women

Solo travel intelligence platform built for women traveling alone in India and beyond.

Wander Women aggregates safety intel, scam reports, stay recommendations, and community knowledge into a single destination guide — surfaced by city, verified by travelers, and updated continuously.

## What it does

- **Trip Intel** — per-city safety guides covering stays, transport, scams, and money
- **Beware Board** — crowdsourced scam and harassment reports mapped by location
- **Buddy Matcher** — find solo women travelers headed to the same destination
- **Community feed** — posts, tips, and Q&A from the traveler community
- **Shop** — curated safety gear and travel essentials with affiliate links
- **Contributor profiles** — attribution for women who write and verify intel

## Stack

Next.js 16 (App Router) · TypeScript · Tailwind CSS v4 · Supabase · shadcn/ui

## Running locally

```bash
cp .env.example .env.local
# fill in your Supabase keys
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
